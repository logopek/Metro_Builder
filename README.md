# Metro Builder
App to create you metro station

 Installation:
 
 ```
 git clone https://github.com/logopek/Metro_Builder/
 ```
 ```
 cd Metro_Builder
 ```
 ```
 pip install -r requirements.txt
 ```
 ```
 python3 main.py
```
